# Сборка защищенного загрузчика лицензий

## Проблема
Прямая компиляция Python → .pyd сложна и требует C компилятора.

## Решение
1. **Источники хранятся в `license_generator/source/`**
2. **Публичная версия получает ОБФУСЦИРОВАННЫЙ код**
3. **Исходники не попадают в публику**

## Процесс:
1. Разработчик редактирует `license_generator/source/_premium_loader_source.py`
2. При создании публичной версии код обфусцируется
3. Публика получает нечитаемый код
4. Генератор лицензий всегда имеет чистые источники

## Автоматизация:
```bash
cd license_generator
python prepare_for_public.py
```

Это скопирует обфусцированный код в `bot_engine/ai/_premium_loader.py`
