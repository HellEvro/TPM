# ГЕНЕРАТОР ЛИЦЕНЗИЙ - ФИНАЛЬНАЯ ВЕРСИЯ

## Архитектура:

### Для разработчика:
- ✅ **Исходники в `source/`** - чистый код для редактирования
- ✅ **Генератор лицензий** - создает .lic файлы
- ✅ **Всё работает** без ограничений

### Для публичной версии:
- ✅ **НЕТ исходников** `source/` - они не копируются
- ✅ **Публика получает `bot_engine/ai/_premium_loader.py`** - с встроенным декодером
- ✅ **Декодер читает только .lic файлы** - не создает их

## Безопасность:
1. **Исходники** остаются ТОЛЬКО у разработчика в `license_generator/source/`
2. **Публичная версия** получает только `_premium_loader.py` с встроенным декодером
3. **Генератор** `generate_license.py` НЕ копируется в публику
4. **Ключи** видны в декодере, но пользователь не может создавать лицензии без генератора

## Использование:

### Генерация лицензии (только у разработчика):
```bash
cd license_generator
python generate_license.py <HWID> <DAYS>
```

### В публичной версии:
- Пользователь получает `.lic` файл
- Система автоматически проверяет через встроенный декодер
- Всё работает!

## Как обновить код:
1. Редактируй `license_generator/source/_premium_loader_source.py`
2. Копируй изменения в `bot_engine/ai/_premium_loader.py`
3. Публикуй в `InfoBot_Public` БЕЗ `license_generator/`!
