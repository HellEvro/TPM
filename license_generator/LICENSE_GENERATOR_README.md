# License Generator

## Назначение
Эта папка содержит инструменты для генерации и управления лицензиями.

**⚠️ ВАЖНО:** Эта папка НЕ копируется в публичную версию!

## Структура:
- `generate_license.py` - генератор лицензий
- `encrypt_core.py` - шифратор ядра AI модулей
- `license_manager.py` - управление лицензиями (копия из InfoBot_AI_Premium)

## Использование:

### Генерация лицензии:
```bash
cd license_generator
python generate_license.py <HWID> <DAYS>
```

### Шифрование ядра:
```bash
python encrypt_core.py
```

## Безопасность:
- Генерация лицензий доступна ТОЛЬКО разработчику
- Клиенты получают только зашифрованные .lic файлы
- Логика расшифровки встроена в основной код (обфусцирована)
