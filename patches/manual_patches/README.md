# Ручные патчи (Manual Patches)

Патчи для применения изменений из последних коммитов, если `git pull` недоступен или возникли конфликты.

## Как применить

### Вариант 1: Применить все патчи по порядку

```bash
cd /путь/к/InfoBot
git am patches/manual_patches/*.patch
```

### Вариант 2: Применить один конкретный патч

```bash
git apply patches/manual_patches/0001-....patch
# или
patch -p1 < patches/manual_patches/0001-....patch
```

### Вариант 3: Через git apply (без коммита)

```bash
for f in patches/manual_patches/*.patch; do git apply "$f"; done
```

## Содержимое патчей (по порядку)

1. **0001** — Единая аналитика для ИИ (ai_analytics_events, API ai-context, ai_summary)
2. **0002** — ИИ смотрит аналитику перед каждой сделкой, пересчитывает по ошибкам
3. **0003** — Фиксация опыта ИИ (ai_experience_snapshot, загрузка при решениях)
4. **0004** — FullAI: модель — основной решатель, аналитика как признаки
5. **0005** — Аналитика: PnL по сделке, колонка «Вывод» в таблице
6. **0006** — Сохранение сделки в bot_trades_history при закрытии через UI

## Рекомендация

Предпочтительно выполнять `git pull` для получения всех обновлений. Патчи — запасной вариант.
